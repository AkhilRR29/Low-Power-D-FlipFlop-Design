# Low Power D Flip-Flop Design

## Project Overview
This project compares different D Flip-Flop architectures for low power VLSI design.

## Implementations
- Conventional D Flip-Flop
- Pulsed D Flip-Flop
- Master-Slave D Flip-Flop
- Reversible Gate D Flip-Flop

## Tools Used
- LTspice (circuit simulation)
- Verilog (digital design)

## Features
- 64-bit register implementation
- Power comparison
- Waveform analysis
